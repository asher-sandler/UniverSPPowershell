General Information

The script sends email notifications to users who have not completed registration in the system

after the last update in the candidate's file and not after 5 days.

Used XML Ini Files

Script uses XML ini files situated in:

  • \\ekeksql00\SP_Resources$\TSS\Reminders\TSS-ARS10.xml
  • \\ekeksql00\SP_Resources$\TSS\Reminders\TSS-ARS11.xml

Used Mail Template Ini File

Script uses Text Template file situated in:

  • \\ekeksql00\SP_Resources$\TSS\Reminders\TSSMailTemplate.txt

Log Files

  • Log files located in "\\ekekfls00\data$\scriptFolders\LOGS\TSSReminder" directory

Email Template

Email template file located in \\ekeksql00\SP_Resources$\TSS\Reminders\TSSMailTemplate.txt directory.

The are custom fields (parameters) collected from SharePoint "Applicants" list: For clarification please contact developer.

Parameters are:

  • "CustName" - Customer name
  • "OpenfilesUpload" - Link to Upload Page
  • "openCancelPage" - Link to Cancel Page

    • No other parameters alowed.
